@Title Ce bogat, ce bogat este harul
@Number 362

1. Ce bogat, ce bogat este harul!
Nu-i cuvant sa-l cuprinda deplin,
Caci din vecii trecuti varsa darul
Peste vecii frumosi care vin. - bis

2. Ce frumos, ce frumos este harul!
Mai frumos ca deplinul frumos;
Cu iubire-si aprinde altarul,
In iubire-l vedem luminos. - bis

3. Ce maret, ce maret este harul
In Hristos, cand, smerit, s-a-ntrupat,
Cand senin a urcat El Calvarul
Si cu Tatal din cer ne-a-mpacat. - bis

4. Cat de larg, cat de larg este harul!
Caci in el pentru veci s-au cuprins
Si dreptatea preasfanta si jarul
Arzator al iubirii nestins. - bis

5. Ce deplin, ce deplin este harul!
E deplin Dumnezeu Preamarit,
Sa-i primesti prin credinta azi darul
Mantuirii, ce El ne-a gatit. - bis
